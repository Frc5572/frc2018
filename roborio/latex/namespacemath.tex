Enforces a wrapping limit on value. 

Wrapping is a constraint in which a minimum is equal to a maximum, and values exceeding either limit \char`\"{}wraps\char`\"{} to the other extremum. An example is angles. The angles 0, and $2\pi$ are equal in angles, so you may enforce a wrapping limit when checking for a value such as $\pi$.
